所有的枚举类型字段必须要有注释，说明每个数据项的用途。

枚举字段【%s】缺少注释信息

Examples:

    public enum TestEnum {
        /**
         * agree
         */
        agree("agree"),
        /**
         * reject
         */
        reject("reject");

        private String action;

        TestEnum(String action) {
            this.action = action;
        }

        public String getAction() {
            return action;
        }
    }
